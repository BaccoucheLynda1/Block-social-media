# Block-social-media
spending more time than usual on social media this isn't just a habit ! its a trap
This UserScript blocks access to common social media domains (Facebook, Instagram, Twitter, TikTok, Snapchat) by stopping the page load and showing a reminder message.
for the use of this code you need to download such extensions (Tampermonkey)
## Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/) in your browser.
2. Click on the script file in this repository.
3. Click **Raw** → Tampermonkey will prompt you to install it.
4. Done! The script will run automatically on blocked domains.
